# Expo App

## Описание
Это приложение, разработанное с использованием Expo и React Native. Оно включает в себя фронтенд и бэкенд части, а также интеграцию с tRPC.

## Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/expo-app.git
   cd expo-app
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```

## Запуск
- Для запуска приложения в режиме разработки:
  ```bash
  npm run start
  ```
- Для запуска веб-версии:
  ```bash
  npm run start-web
  ```

## Деплой
Приложение настроено для деплоя на Vercel. Для деплоя выполните следующие шаги:
1. Убедитесь, что у вас установлен Vercel CLI:
   ```bash
   npm install -g vercel
   ```
2. Выполните деплой:
   ```bash
   vercel
   ```

## Переменные окружения
Убедитесь, что файл `.env` не попадает в репозиторий. В `.gitignore` это уже учтено.
